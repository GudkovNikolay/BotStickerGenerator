# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
python -m venv venv
source venv/bin/activate  # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Шаг 2: Настройка конфигурации

1. Создайте файл `.env` в корне проекта
2. Скопируйте содержимое из `.env.example` и заполните:

```env
BOT_TOKEN=ваш_токен_от_BotFather
OPENAI_API_KEY=ваш_openai_api_ключ
```

Минимально необходимые переменные:
- `BOT_TOKEN` - получите у @BotFather в Telegram
- `OPENAI_API_KEY` - получите на https://platform.openai.com/api-keys

## Шаг 3: Запуск бота

```bash
python bot.py
```

Бот запустится и начнет отвечать на команды в Telegram.

## Первые команды

1. Откройте вашего бота в Telegram
2. Отправьте `/start`
3. Отправьте `/generate` и следуйте инструкциям
4. Введите описание стикеров (например: "Кот в космосе")
5. Дождитесь генерации и получите стикеры!

## Структура проекта

- `bot.py` - главный файл запуска
- `handlers.py` - обработчики команд и сообщений
- `database.py` - модели базы данных
- `db_service.py` - сервис для работы с БД
- `image_generator.py` - генерация изображений через DALL-E 3
- `sticker_processor.py` - обработка изображений в стикеры
- `payment_service.py` - сервис платежей (заглушка для MVP)
- `config.py` - конфигурация приложения
- `utils.py` - вспомогательные функции

## Важные замечания

- Для работы требуется активный интернет-соединение
- Генерация изображений через DALL-E 3 платная (проверьте тарифы OpenAI)
- База данных SQLite создается автоматически при первом запуске
- Логи сохраняются в файл `bot.log`

## Решение проблем

**Ошибка "Ошибка загрузки конфигурации":**
- Убедитесь, что файл `.env` существует и содержит все необходимые переменные

**Ошибка при генерации изображений:**
- Проверьте, что `OPENAI_API_KEY` правильный и имеет доступ к DALL-E 3
- Убедитесь, что на счету OpenAI есть средства

**Бот не отвечает:**
- Проверьте, что `BOT_TOKEN` правильный
- Убедитесь, что бот запущен (`python bot.py`)
- Проверьте логи в файле `bot.log`
